# Robot SEO

A modern Next.js website for Robot SEO consultancy services - making websites maximally legible to AI systems while maintaining traditional SEO excellence.

## About Robot SEO

Robot SEO is the practice of making websites maximally legible, trustworthy, and citable to AI systems (such as search engines, AI assistants, and automated agents) while also maintaining best-in-class traditional SEO. This specialized approach addresses the technical and structural requirements that AI systems use to understand, evaluate, and cite web content.

## Tech Stack

- **Next.js 16.1.6** - React framework with App Router
- **TypeScript** - Type safety and enhanced developer experience
- **Tailwind CSS** - Utility-first CSS framework
- **ESLint** - Code linting and formatting

## Getting Started

### Development Server

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
npm start
```

### Linting

```bash
npm run lint
```
